# Smart-Food-Traker
Smart Food Saver is a web-based food waste management application that helps users track their food consumption, reduce waste, and contribute to environmental sustainability. Built with vanilla HTML, CSS, and JavaScript, this application provides an intuitive interface for managing food inventory, monitoring expiry dates, and discovering recipes for leftovers, etc.

Key Features :

1. User Authentication - Secure login and registration system with personalized user accounts
2. Dashboard Analytics - Track food waste statistics, money saved, and COâ‚‚ impact reduction
3. Expiry Alerts - Real-time browser notifications for items nearing expiration
4. Recipe Suggestions - 10+ leftover recipes with detailed instructions and tips
5. Donation Centers - Locate nearby food banks and donation opportunities
6. Visual Charts - Interactive graphs showing waste trends over time (Chart.js)
7. Gamification - Level system with points and achievements for motivation
8. Data Export - Download your food logs in JSON or CSV format
9. Dark Mode - Toggle between light and dark themes
10. Responsive Design - Works seamlessly on desktop, tablet, and mobile devices.
